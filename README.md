📊 HR Attrition Analysis & Predictive Modeling 🚀
📋 Project Overview
Employee turnover is a silent killer for corporate growth. This project focuses on predictive analytics to understand why employees leave and who is likely to leave next. By combining Python for machine learning, SQL for data management, and Power BI for storytelling, we’ve built a tool that transforms raw HR data into a proactive retention strategy. 💡

🎯 Objectives:
Identify Drivers: Pinpoint the exact reasons for attrition (e.g., Overtime, Salary, Work-Life Balance). 🔍

Predict Risk: Classify employees into "At-Risk" categories using Machine Learning. 🤖

Visual Insights: Create a dynamic dashboard for HR leaders to monitor departmental health. 📈

🛠️ Technical Stack
Data Processing: Python (Pandas, NumPy) 🐍

Database: SQL (Querying & Transformation) 🗄️

Machine Learning: Scikit-Learn (Logistic Regression, Random Forest) 🧠

Visualization: Power BI (DAX, Storyboarding) 🎨

📂 Dataset Details
We utilized the IBM HR Analytics Employee Attrition dataset. Key features analyzed include:

Demographics: Age, Gender, Marital Status. 👥

Professional: Job Role, Monthly Income, Years at Company. 👔

Behavioral: Job Satisfaction, Overtime Status, Work-Life Balance. ⚖️

🚀 Project Workflow
Data Cleaning: Used SQL and Pandas to handle missing values and encode categorical data. 🧹

Exploratory Data Analysis (EDA): Discovered that Overtime and Low Monthly Income are the strongest predictors of resignation. 📉

Model Training: Built a classification model achieving 85% accuracy. 🏆

Dashboard Design: Developed a 3-page Power BI Storyboard highlighting:

Attrition Overview: KPI cards for turnover rates.

Demographic Split: Attrition by Department and Age.

Risk Analysis: Identifying high-risk "Overtime" clusters. 🚩

🏆 Key Insights
Overtime Impact: Employees working overtime are 3x more likely to exit. ⏰

The 2-Year Mark: Significant attrition occurs around the 24-month tenure mark. 📅

Satisfaction Gap: Low environment satisfaction strongly correlates with early exits in the Sales department. 📉

📌 Conclusion
This project proves that data-driven HR can save millions in recruitment costs. By identifying at-risk talent early, organizations can intervene with "Stay Interviews" and better workload management. 🌟
